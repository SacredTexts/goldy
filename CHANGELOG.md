# Changelog

All notable changes to GOLDY will be documented in this file.

## [1.0.0] - 2026-03-01

### Added
- Initial release as a standalone repo
- Universal installer (`install.sh`) for macOS, Linux, and WSL2
- Global-only installation (no per-project install)
- `/goldy` — Gold Standard planning with stack-aware decisions
- `/goldy-loop` — Phase loop execution with checkpoints and resume
- 21 Python scripts (pure stdlib, zero pip dependencies)
- 13 stack profiles (React, Next.js, Vue, Svelte, etc.)
- Gold Standard plan template
- Makefile with test, lint, install, update, uninstall targets
- CONTRIBUTING.md with development workflow
