# Command Templates

These files are **reference templates** only. They contain the original macOS paths.

The actual slash commands installed to `~/.claude/commands/` are **generated by `install.sh`** with paths resolved for your system. Do not manually copy these files — run `bash install.sh` instead.

If you modify the command behavior (frontmatter, description, instructions), update these templates, then re-run `install.sh` to apply.
